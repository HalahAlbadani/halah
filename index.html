Projects

Sarcasm Detection

Technologies/Libraries Used :

Python, NLTK, TextBlob, scikit-learn, numpy

My Undergraduate college (PESIT Bangalore South Campus) has a lecture recording system in place. This system records the contents of the lecture in an audio-video format along with a transcript of the audio. The aim of this project is to create a set of notes from the transcript that contain only the important points of the topic and remove the non-essential classroom talk. In this project, my team and I have created a program which can scan a given document for sarcasm and analogies, and extract them . The project relies on Natural Language Programming and Machine Learning to train the machine to detect these automatically without human intervention. With this automated system in place, students may use the transcripts that are generated.
